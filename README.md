## 🎨 Premium Version Features

### **1. Full 3D Rotating Text**
- Continuous smooth 360° rotation on Y-axis
- 3D cube with 4 faces: "We Design", "We Brand", "We Print"
- Professional depth and shadow effects
- Always readable, no distortion

### **2. Multi-Page Architecture**
- **5 Independent Pages** (no vertical scrolling)
  1. **Home** - 3D hero, statistics, CTAs
  2. **About** - 4 value propositions
  3. **Services** - 6 service cards
  4. **Payment** - M-Pesa integration (Till: 4136540)
  5. **Feedback** - Customer suggestions
- Each page = 100% viewport height & width

### **3. Horizontal Navigation**
- Click menu items to slide pages smoothly
- Keyboard arrows (← →) to navigate
- 0.8s cubic-bezier transition
- Active link highlighting in orange

### **4. Book-Flip Transitions**
- 3D perspective rotation on page entrance
- Premium, realistic page-turning effect
- 0.8s ease-out animation

### **5. Professional Design**
- **Colors:** White, Black, Orange, Green only
- **Typography:** Garamond (headers), Times New Roman (body)
- **Base Font Size:** 12px
- **Responsive:** 320px - 1920px
